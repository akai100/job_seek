# auto & decltype 类型推导
+ auto
  基于变量的**初始化表达式**推导类型，会**忽略**表达式的顶层const和引用；
+ decltype
  基于**给定表达式**的完整类型推导，**保留**顶层const和引用；
差异：
+ ```auto```必须待初始化器（否则无法推导类型）
+ ```decltype```无需初始化器，可直接声明变量
