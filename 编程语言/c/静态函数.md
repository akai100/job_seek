## 静态函数
用 static 修饰的函数，定义为静态函数，静态函数仅在其声明的文件中可见，不能被其他文件所用。

## 定义在头文件的静态函数
// a.h
```C
#include <stdio.h>

static int sint_t = 1;

static void tt()
{
    printf("%d \n, sint_t);
}
```
// b.c
```C
#include "a.h"

int main() {
    tt();
   func();
   return 0;
}
```
// c.c
```C
#include "a.h"

void func() {
    tt();
}
````
两个源文件引用的同一头文件中的静态变量/函数，在两个文件中的内存区域中独立存在。
